## weapp-plugin-demo

有赞微商城所有小程序插件的演示demo

![Demo 演示](https://img.yzcdn.cn/upload_files/2018/07/26/FtcFpmMGvjMLQAIdURcyeYMnx6Mt.jpg?imageView2/2/w/300/h/300)

## 如何在本地预览demo

1. 在根目录下运行

```shell
npm install
npm run dev
```

2. 打开`微信开发者工具`，'本地小程序项目 - 添加项目'，把 `dist` 目录添加进去就可以预览示例demo了。
